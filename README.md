#Weather app
The Weather App is a simple Python-based application that fetches and displays real-time weather information for any city using a public API. It demonstrates how to work with APIs, JSON data, and Python programming to build a functional and interactive tool.

#Features:

Search weather by city name

View temperature, humidity, and weather conditions

Easy-to-use command-line interface

#Technologies Used:

Python – Programming language for the app logic

Requests Library – To fetch data from the API

JSON – For parsing and handling API responses

OpenWeatherMap API – Source of live weather data
